# RonaScript
RonaScript is a statically typed, object oriented, scripting language.

## Features
* Identifier aliasing
* Automatic garbage collection
* Static typing

## Status
Development of the language is currently underway, as such, the following considerations should be made:
* main.cpp is not included in the repository.
* The project may not work as expected if you try to build/run it.
* Significant changes to the design may be made at any time.
