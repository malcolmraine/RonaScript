# RonaScript
RonaScript is a statically typed, object oriented, scripting language inspired by PHP, Python, JS, and more.

## Features
* Identifier aliasing
* Option for range limited variables (future)
* Automatic garbage collection
* Static typing

## Status
Development of the language is currently underway. The first release of the implementation will be done entirely in Python. Further releases will incorporate more C or C++ components for runtime improvements.
